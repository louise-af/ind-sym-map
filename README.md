# ind-sym-map

1. Node modules to install with ```npm install```:
* express
* bcrypt
* passport
* express-flash
* express-session
* method-override

2. Copy example.env and name it .env

3. Create your database file with 
```npm run initDB```

4. To start server:
```npm run devStart```
runs at localhost:3000

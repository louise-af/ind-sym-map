# ind-sym-map

Node modules to install with ```npm i```:
* express
* bcrypt
* passport
* express-flash
* express-session
* method-override

Copy example.env and name it .env

Create directory db
```mkdir db```

Create your database file with 
```npm run initDB```

To start server:
```npm run devStart```
runs at localhost:3000
